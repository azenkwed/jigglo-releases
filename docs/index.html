What Jigglo does

Jigglo runs in your Windows system tray and makes small, configurable mouse movements in the background so your machine doesn't go idle, lock, or sleep while you're:

  • Watching videos, trainings, or livestreams
  • Reading or reviewing long documents
  • Monitoring long-running jobs or remote sessions

You control how often the mouse jiggles, how far it moves, and whether it should auto-start when you've been idle for a few minutes.

Key features

  • Lightweight tray app with a simple menu
  • Intervals: 15 seconds, 30 seconds, 1, 2, or 4 minutes
  • Jiggle sizes: small, medium, large
  • Optional auto-start when idle (4‑minute threshold)
  • Optional launch at login for your user account
  • Per-user settings stored in %APPDATA%\Jigglo\settings.json

System requirements

  • Windows 10 or later (64‑bit)
  • .NET Framework 4.8 (normally already present on Windows 10/11)

How to install

  1. Download the ZIP from GitHub Releases.
  2. Right‑click the ZIP and choose Extract All….
  3. Extract to a folder you trust (for example C:\Users\<you>\Apps\Jigglo).
  4. Run Jigglo.exe.

Trust & security

  • No network access: Jigglo does not talk to the internet or send telemetry.
  • Only touches your mouse cursor, a per‑user settings file, and (optionally) the Run registry key to launch at login.
  • Not currently code‑signed, so SmartScreen may show a warning on first run—only bypass it if you downloaded from the official release and you trust it.
  • You can verify the ZIP integrity using the SHA‑256 hash above.

Work laptops & IT policies

By design, Jigglo can interfere with power‑saving and lock‑screen policies. On work, school, or otherwise managed devices:

  • Treat Jigglo like any other third‑party utility.
  • Follow your organization's policies around screen locking, idle timeouts, and installing software.
  • Do not use Jigglo to deliberately bypass security, compliance, monitoring, or time tracking requirements.
  • When in doubt, ask your IT team before installing or running Jigglo.
